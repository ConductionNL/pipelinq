# Public Intake Forms

## Problem
No public form capability exists. Organizations cannot capture leads or service requests from their website without requiring visitors to create accounts. 61% of government tenders require formulieren/intake.

## Proposed Solution
Build a form builder for creating embeddable HTML forms that create contacts and leads in Pipelinq upon submission. Support iframe and JavaScript snippet embedding, AVG-compliant forms, NL Design System styling, and spam protection.

## Impact
- No-code form builder in admin settings
- Public form rendering engine (no auth required)
- Embed code generation (iframe and JS snippet)
- New form, formField, formSubmission schemas
- Spam protection (honeypot + optional CAPTCHA)
