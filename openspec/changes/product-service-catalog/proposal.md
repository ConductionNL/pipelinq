# Product & Service Catalog (PDC)

## Problem
No government-standard product and service catalog exists. 65% of tenders require a PDC for citizen-facing portals. Municipalities need UPL-compliant product listings, multilingual content for EU SDG compliance, and integration with IPDC.

## Proposed Solution
Build a PDC module as a dedicated register with UPL/IPDC-compliant product schema, publication lifecycle, target audience classification, pricing, multilingual content, and a public read-only API for municipal websites.

## Impact
- New `pdc` register with product, category, tariff schemas
- UPL reference list integration
- Public API for citizen portal integration
- Multilingual content support for SDG compliance
- Separate from the existing CRM product catalog (different domain)
