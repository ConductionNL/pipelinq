# Kennisbank

## Problem
No knowledge base exists for KCC agents. Agents cannot search for articles, FAQs, or procedures to answer citizen questions. First-call resolution suffers without readily available reference information.

## Proposed Solution
Build a knowledge base with article management (create, publish, archive, version), category hierarchy, full-text search, zaaktype linking, and agent feedback. Articles stored as OpenRegister objects.

## Impact
- New `kennisartikel` and `kenniscategorie` schemas
- New knowledge base views and components
- Full-text search integration
- Category management in admin settings
